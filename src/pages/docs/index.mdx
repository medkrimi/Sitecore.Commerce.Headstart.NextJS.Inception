# Introduction
 
Welcome to Sitecore Commerce Seller App! This is a basic implementation of the Sitecore Commerce using the Order Cloud Javascript SDK. 
You can use it as a starting point to discover, undrestand and learn Sitecore Commerce.
This showcase different marketplace scenarios B2B, B2C, B2B2C.   


## What is Sitecore Commerce Seller App?

A **simple**, **powerful** and **flexible** Commerce Seller App built on top of Sitecore Order Cloud Javascript SDK using NextJS and Chakra UI.
