# Manage Permission
Sitecore Commerce Seller App support Roles and Permissions defined into the Order Cloud Commerce instance. 
All Roles and Permission are defined in: 
```
\src\lib\constants\app-permissions.config.ts
```

## Update Role and Permissions
1. Go to your Sitecore Order Cloud Commerce instance 
2. Click on Security Profile 
3. Create new Security Profile and assign roles
4. Go to Admin User Groups and add a new User Groups (To be able to assign it to any specific user if needed)

The current Admin user has the ```full access``` role.
